# 蚂蚁巢穴 - Godot 4.x 2D项目

## 项目结构

```
godot_project/
├── project.godot          # 项目配置（1080x1920分辨率、输入绑定）
├── icon.svg               # 临时图标
├── README.md              # 使用说明
├── scenes/
│   ├── game_manager.tscn # 游戏管理器（基地/战场切换）
│   ├── base_camp.tscn    # 蚁巢基地界面
│   ├── main.tscn         # 战场主场景
│   ├── player.tscn       # 玩家蚂蚁
│   ├── enemy.tscn        # 敌人甲虫
│   ├── extraction_point.tscn  # 撤离点
│   └── resource_pickup.tscn  # 残骸拾取物
└── scripts/
	├── game_manager.gd    # 游戏状态管理、场景切换
	├── base_camp.gd       # 基地逻辑、生产队列、物资管理
	├── main.gd           # 战场逻辑、撤离/物资管理
	├── player.gd          # 玩家控制、蓄力攻击
	├── enemy.gd           # 敌人AI
	├── extraction_point.gd # 撤离点逻辑
	└── resource_pickup.gd # 残骸拾取
```

## 使用方法

1. 下载并安装 [Godot 4.2+](https://godotengine.org/download)
2. 打开Godot，点击"Import"导入项目
3. 选择 `project.godot` 文件
4. 点击"Run"运行项目

## 操作说明

### 基地界面
- 查看蚁巢状态
- 查看物资存储
- 管理生产队列
- 点击"进入战场"开始战斗

### 战场操作
| 按键 | 功能 |
|------|------|
| W/↑ | 向上移动 |
| S/↓ | 向下移动 |
| A/← | 向左移动 |
| D/→ | 向右移动 |
| Q | 旋转攻击方向（向左） |
| E | 旋转攻击方向（向右） |
| 空格 | 按住蓄力，松开释放攻击 |
| ESC | 退出游戏 |
