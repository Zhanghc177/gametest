# 地下蟻巢视觉哲学：Subterranean Emergence

## 设计哲学

**核心理念**：地下世界的生物发光美学与有机力量

地下巢穴是一个神秘而充满生命力的世界。蚂蚁作为社会性昆虫，其视觉语言应当传达出：
1. **有机复杂性** - 巢穴内部的层次感和生物形态
2. **生物发光** - 黑暗中发出的微光，标识重要元素
3. **力量与凝聚力** - 蚂蚁群体的协调之美

## 视觉风格方向

### 1. 生物发光美学 (Bioluminescent Aesthetic)
- 核心：从深海生物和萤火虫获取灵感
- 实现：资源、撤离点发出柔和的光晕
- 玩家蚂蚁边缘可添加微弱发光效果
- 营造"黑暗中闪烁生命之光"的氛围

### 2. 有机几何 (Organic Geometry)
- 蚂蚁身体用圆角矩形而非精确几何
- 障碍物采用不规则边缘的岩石形态
- 粒子效果使用圆润的形状，避免锐角

### 3. 层次感光影 (Layered Lighting)
- 背景保持深色，但添加微妙的纹理
- 前景元素有清晰的光源方向
- 重要元素（玩家、敌人、资源）有光晕分离层次

## 元素设计规范

### 玩家蚂蚁
```
建议：
- 头部：深褐色圆角矩形，带两个小眼睛
- 身体：主色橙褐，略带渐变
- 腹部：略深的橙褐色，椭圆形
- 腿部：可选添加，作为装饰
- 边缘发光：表示生命/能量
```

### 敌人（甲虫类）
```
建议：
- 更宽的甲壳形态
- 蓝灰色调，冷峻
- 红色复眼，闪烁效果
- 甲壳有光泽渐变
```

### 障碍物（岩石）
```
建议：
- 不规则多边形
- 灰色系，带深色阴影
- 边缘柔和，不锐利
- 可添加裂缝纹理
```

### 资源
```
当前：橙色菱形（已有基础）
建议：
- 保持菱形核心形状
- 添加柔和光晕
- 悬浮动画（上下浮动）
- 收集时有粒子爆发
```

### 撤离点
```
当前：橙色菱形+圆形范围（已有基础）
建议：
- 菱形保持，作为核心
- 圆形区域有脉冲动画
- 添加方向性指示（箭头）
- 撤离时有传送光效
```

## 动效与粒子

### 粒子效果清单
1. **资源收集** - 橙色彩粒子向外爆发
2. **撤离传送** - 螺旋光效，菱形分解
3. **攻击命中** - 红色冲击波
4. **蚂蚁移动** - 腿部扬起的小尘土

### 动画节奏
- 资源浮动：缓慢正弦波动（2-3秒周期）
- 光晕脉动：温和呼吸感（1.5秒周期）
- 状态转换：快速弹跳（0.3秒）
